# x2p: anything-to-prompt

Convert arbitrary digital contexts (web pages, codebases, PDFs, images, dashboards) into structured, token-efficient prompts for LLMs.

## What it is

`x2p` is a plugin-driven Rust framework with two surfaces:

- **CLI** (`x2p`): capture and render prompts from the terminal
- **Browser Extension** (Chromium + Firefox, MV3): capture live web UI context
